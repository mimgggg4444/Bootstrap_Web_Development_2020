# Bootstrap_Web_Project

"Bootstrap_Web_Project"는 Bootstrap을 활용하여 웹사이트를 개발하는 프로젝트입니다. 이 프로젝트에서는 Bootstrap의 다양한 기능을 활용하여 반응형 웹사이트를 제작하였습니다.

## 주요 기능:

1. **Bootstrap 기반 웹사이트 제작**: 이 프로젝트의 주요 목적은 Bootstrap을 사용하여 웹사이트를 제작하는 것입니다.
2. **반응형 웹 디자인**: 이 웹사이트는 다양한 디바이스에서 잘 보이도록 반응형 웹 디자인을 적용하였습니다.

이 프로젝트의 코드는 각 폴더에 정리되어 있습니다. 'adminPage', 'css', 'js' 폴더에는 이 프로젝트에서 개발한 주요 코드와 관련된 파일들이 포함되어 있습니다.
